Multi-agent and multi-channel widget to offer your customers the easiest way to contact from your website, in just 2 clicks!


== Description ==

Lead2Team plugin for Wordpress allows you to add a list of agents or departments to your website. Users can contact with them on different channels like WhatsApp, phone call, email, and many others.


✅ WHY USE LEAD2TEAM WIDGET?

➡ More web conversion.
Increase leads generated by over 40%.

➡ Lead routing.
Automatically route requests to the right departments or agents.

➡ Better user experience.
All your services and contact options in one window, 2 clicks away!

➡ Build loyalty direct from your website.
Offer closer communication. People buy from people.

➡ Measure which channels and agents receive more interactions. 
Access lead reports and rankings.


⚡️ More than a widget ⚡️ 
Now, includes 3 new formats you can use for free. Discover more [here](https://lead2team.com)



✅ EASY CONFIGURATION IN LESS THAN 15 MIN.


▪ Create agents profile and upload their photo.

▪ Add the channels and links to each agent.

▪ You can customize the look a feel.

▪ Classify agentes by teams.

▪ Filter the widget code and show in every page a different team.



== Installation ==


✅ Installing the Lead2Team pluguin for WordPress just takes a few minutes.


1. **Visit** Plugins > Add New
2. **Search** for "Lead2Team"
3. **Activate** Lead2Team from your Plugins page
4. **Click** on the new menu item "Lead2Team" under Tools
